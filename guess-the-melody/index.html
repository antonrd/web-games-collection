Musical hearing trainer

Use this tool to generate short melodies. Listen to the melodies and try to reproduce them on a music instrument, like a piano, to train your musical hearing ability. When done, reveal the melody notes to check if you've guessed it right.

The page opens with a default generated melody. You can select to generate a new random melody or select one from a pre-defined set. If a random melody is generated, you can select the number of notes and the scale to use. You can also leave the scale to be randomly selected. The notes can played using sampled notes or ones generated on the fly.

Once ready with the melody settings:

  • Click the "Generate Melody" button, which will show the notes of the melody without revealing them.
  • You can then click the "Play Melody" button to listen to the melody.
  • You can stop the playing with the button "Stop Melody".
  • When you want to see the notes, click the "Reveal Melody" button to have them appear in the notes boxes.
